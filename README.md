# Gemini Poem Generator 🖋️

A simple Python script that uses the Google Gemini API to generate custom poetry. 

## Why I Built This
I built this project to learn how to connect to an AI API using Python, and to generate some comforting words
during a tough time or just to put feelings into words.

## How to Run It
1. Clone this repository.
2. Install the required libraries: `pip install google-genai python-dotenv`
3. Create a `.env` file and add your Google Gemini API key: `GEMINI_API_KEY="your_key_here"`
4. Run the script: `python poem_generator.py`
